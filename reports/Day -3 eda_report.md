DAY 3 – EXPLORATORY DATA ANALYSIS (EDA)

NAV Analysis

1. NAV history dataset contains 46,000 records across 40 mutual fund schemes.
2. NAV data spans from 03-Jan-2022 to 29-May-2026.
3. Date column converted from string to datetime format.
4. Interactive NAV trend visualization created using Plotly.
5. Market bull run period (2023) and correction period (2024) highlighted.

AUM Analysis

1. AUM dataset contains data for 10 fund houses.
2. AUM data covers the period from 2022 to 2025.
3. Date column converted to datetime format.
4. Grouped bar chart created for fund house-wise AUM comparison.
5. SBI Mutual Fund maintained the highest AUM among fund houses.

SIP Analysis

1. SIP dataset contains 48 monthly observations from Jan 2022 to Dec 2025.
2. Monthly SIP inflow trend visualized using a time-series chart.
3. SIP inflows showed an overall increasing trend.
4. Highest SIP inflow recorded in December 2025.
5. Maximum SIP inflow observed: ₹31,002 crore.
6. Active SIP accounts increased steadily during the analysis period.
